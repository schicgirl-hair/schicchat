# SchicChat Client Pro Polished

Version client uniquement, prête pour GitHub Pages.

## Améliorations
- Site plus stable et responsive.
- Polices réduites pour une lecture plus élégante.
- Chat agrandi pour mobile et desktop.
- Header du chat retiré pour laisser plus d’espace.
- Liens Selar transformés en boutons cliquables.
- Message “diagnostic déjà utilisé” amélioré.
- Version autonome : pas de Google Form, pas de PDF, pas de backend.

## Publier sur GitHub
Mets ces fichiers à la racine du repo :
- index.html
- style.css
- script.js
- config.js

Puis GitHub Pages : branch main, folder /root.

## Important
La limite “une seule utilisation” utilise localStorage. Elle bloque le même navigateur, pas une personne partout.
